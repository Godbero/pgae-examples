This example includes a version of the Google App Engine Helper for
Django, an open source project released under the Apache 2.0 license.
The original license is in the file named "COPYING".

You can download the latest version of the Helper starter project
here:
  http://code.google.com/p/google-app-engine-django/

To run the sample, you must have Django 1.1 installed with your Python
distribution.  You can get Django 1.1 here:
  http://www.djangoproject.com/
