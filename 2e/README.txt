This collection contains examples from the book _Programming Google
App Engine_, 2nd edition, by Dan Sanderson, originally published in
October 2012.

You can get the latest version of this collection from the website for
the book:
  http://www.dansanderson.com/appengine/

You can also get the code using Mercurial, as well as report an error
or omission, at the collection's project site:
  http://code.google.com/p/pgae-examples/

See the README file for your language for more information:
  README_java.txt
  README_python.txt
